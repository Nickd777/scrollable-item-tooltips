# Scrollable Item Tooltips

Client-side Fabric mod for **Minecraft 26.1.2** that lets you **scroll long item tooltips** with the
mouse wheel instead of having them run off the screen. Fabric API not required.

## Install

Install Fabric Loader for `26.1.2`, then drop `scrollabletooltips-1.0.0.jar` into `.minecraft/mods`.

## Config

Auto-created at `.minecraft/config/scrollabletooltips.json`: `scrollSpeed`, `invertScroll`,
`showScrollbar`, `edgeMargin`, `enabled`.

## Build

`./gradlew build` (JDK 25) → `build/libs/scrollabletooltips-1.0.0.jar`.
